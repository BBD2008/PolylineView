# PolylineView
可以用于大量数据处理的“折线图”，内存和CPU占用都很好。

##动画演示
实际使用非常流畅，gif图受帧率限制无法流畅演示。

![image](https://github.com/BBD2008/PolylineView/raw/master/Image/naodianbo.gif)

##导入方式

把WaveView文件夹拖到项目中即可使用。具体使用方式见“折线demo”。

##使用方式

```Objective-C

//核心方法
- (void)refreshWaveWithValue:(float)value;

```
